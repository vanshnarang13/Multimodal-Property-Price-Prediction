Zoom Level Experiment Results
==================================================

Zoom levels tested: [18, 19, 20, 21]
Number of locations: 3
Image size: 640x640

Zoom Level Guide:
  18 - Regional view, shows neighborhood area
  19 - Detailed view, individual buildings visible
  20 - Very detailed, can see building features
  21 - Maximum detail, highest resolution

Each location_X folder contains the same coordinates
at different zoom levels for comparison.
